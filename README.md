# Esercizi Repository

This repository contains a collection of coding exercises, projects, and study materials across various programming languages and topics.

## Directory Structure

The repository is organized into the following categories:

- **`codewars`**: Solutions to algorithmic challenges from Codewars (Python).
- **`cybersecurity`**: Exercises and notes related to cybersecurity.
- **`database`**: Database schemas, queries, and related exercises.
- **`flask`**: Web applications and experiments using the Flask framework (Python).
- **`java`**: Java programming exercises.
- **`personal`**: Personal projects and scripts.
- **`python`**: General Python scripts, exercises, and learning materials.
- **`react`**: React.js frontend applications and components.
- **`web_development`**: Fundamental web development exercises (HTML, CSS, JavaScript), including DOM manipulation and Bootstrap.
- **`software_design`**: Software engineering design patterns, UML diagrams, and project planning exercises.
- **`scratchpad`**: Temporary files and simple tests.

## Technologies

- **Languages**: Python, Java, JavaScript, HTML, CSS, SQL.
- **Frameworks/Libraries**: Flask, React, Bootstrap.
- **Tools**: Git, VS Code.

## Getting Started

Navigate to any directory to explore specific exercises. Most Python scripts can be run directly, while web projects may require installing dependencies (e.g., `npm install` for React, `pip install` for Flask).
